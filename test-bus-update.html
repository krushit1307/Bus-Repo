Bus Update Test

Open browser console and try updating a bus to see detailed logs.

Steps to debug:

  1. Open browser developer tools (F12)
  2. Go to Console tab
  3. Try editing and saving a bus
  4. Check console for detailed error messages
  5. Check Supabase dashboard to see if update appears

Common issues:

  • RLS policies blocking updates
  • ID format mismatch (string vs number)
  • Missing environment variables
  • Network connectivity issues
